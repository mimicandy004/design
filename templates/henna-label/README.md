# 헤나 염색약 라벨 공통 템플릿

이 폴더는 헤나 염색약 라벨 디자인을 만들 때 재사용할 공통 기준을 담습니다. 특정 업체 전용 문구나 이미지가 아니라, 여러 업체에 반복 적용할 수 있는 구조와 스타일 원칙만 보관합니다.

## 사용 범위

- 헤나 염색약 라벨
- 헤어 컬러 제품 라벨
- 미용 재료 파우치 또는 박스 라벨
- 컬러칩이 필요한 제품군 라벨

## 재사용할 기본 요소

- 브랜드 영역
- 제품명 영역
- 선택 컬러 카드
- 컬러 라인업 영역
- 인증/보조 정보 영역
- 용량 표기 영역
- 파우치 또는 박스 목업 적용 방식

## 업체별로 바꿔야 할 내용

- 브랜드명
- 서브 브랜드명
- 제품명
- 컬러 번호
- 컬러명
- 용량
- 인증 문구
- 색상표
- 참고 이미지
- 로고 파일

## 주의 사항

- 인증 문구는 실제 사용 가능 여부를 확인해야 합니다.
- 검증되지 않은 효능 표현은 사용하지 않습니다.
- 특정 업체의 시안 이미지를 그대로 템플릿으로 쓰지 않습니다.
