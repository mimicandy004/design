# 디자인 스타일 제안서 초안

이 문서는 특정 브랜드의 공식 디자인 시스템이 아니라, 앞으로 디자인 제작을 진행할 때 참고하기 위한 스타일 제안서입니다. 피그마 스타일에서 느껴지는 깔끔한 편집감, 큰 여백, 선명한 타이포그래피, 부드러운 컬러 블록의 인상을 참고하되, 특정 서비스명, 로고, 공식 사례는 사용하지 않습니다.

## 1. 전체 방향

전체 디자인은 흑백을 기본으로 하고, 중요한 구간에만 큰 파스텔 컬러 블록을 사용하는 방향을 제안합니다. 기본 화면은 단정하고 조용하게 유지하고, 메시지를 강조해야 하는 부분에서는 넓은 색상 면과 큰 글자를 사용해 시선을 끌어줍니다.

이 스타일은 로고, 배너, 라벨디자인, 패키지 디자인, 상세페이지, SNS 이미지처럼 다양한 결과물에 적용하기 좋습니다. 특히 제품이나 서비스의 장점을 빠르게 보여주고, 복잡한 설명을 보기 쉽게 정리하는 데 적합합니다.

## 2. 핵심 인상

- 깔끔하고 현대적인 인상
- 흑백 중심의 단정한 화면
- 큰 제목과 넉넉한 여백
- 부드러운 파스텔 컬러 블록
- 둥근 버튼과 부드러운 모서리
- 복잡한 장식보다 정보 전달을 우선하는 구성

## 3. 색상 방향

### 기본 색상

- **검정**: 제목, 본문, 주요 버튼, 강조 문구에 사용합니다.
- **흰색**: 기본 배경으로 사용해 정보가 깔끔하게 보이도록 합니다.
- **연한 회색**: 구분선, 입력 영역, 보조 배경에만 제한적으로 사용합니다.

### 강조 색상

강조 색상은 작은 포인트보다 넓은 컬러 블록으로 사용하는 것을 권장합니다. 화면 전체가 복잡해지지 않도록 한 화면에는 한 가지 강조 색상만 중심으로 사용합니다.

- **라임 계열**: 활기, 성장, 친근함을 표현할 때 사용합니다.
- **라일락 계열**: 부드러움, 창의성, 감각적인 분위기를 표현할 때 사용합니다.
- **크림 계열**: 따뜻함, 안정감, 생활감 있는 제품에 사용합니다.
- **민트 계열**: 깨끗함, 신선함, 건강한 인상을 줄 때 사용합니다.
- **핑크 계열**: 귀여움, 감성, 선물용 이미지를 강조할 때 사용합니다.
- **코랄 계열**: 밝고 적극적인 분위기, 이벤트나 프로모션에 사용합니다.
- **네이비 계열**: 고급감, 신뢰감, 강한 대비가 필요할 때 사용합니다.

## 4. 글자 스타일

제목은 크고 분명하게, 본문은 읽기 쉽게 구성합니다. 너무 많은 서체를 섞기보다 하나의 기본 서체 안에서 굵기와 크기만 조절하는 방향이 좋습니다.

### 추천 방식

- 큰 제목은 짧고 강하게 작성합니다.
- 본문은 쉬운 말로 자연스럽게 씁니다.
- 작은 라벨이나 카테고리 문구는 영문 대문자 스타일보다 한글에 맞게 짧은 단어로 정리합니다.
- 가격, 혜택, 제품명, 핵심 문구는 굵기 차이로 구분합니다.

### 예시 역할

| 역할 | 사용 위치 | 권장 느낌 |
|---|---|---|
| 큰 제목 | 상세페이지 상단, 배너, 패키지 앞면 | 크고 선명하게 |
| 중간 제목 | 섹션 제목, 제품 특징 제목 | 짧고 또렷하게 |
| 본문 | 설명문, 사용 방법, 안내 문구 | 읽기 쉽게 |
| 보조 문구 | 주의사항, 작은 설명, 라벨 하단 | 작지만 선명하게 |
| 버튼 문구 | 구매하기, 문의하기, 자세히 보기 | 짧고 행동 중심으로 |

## 5. 레이아웃 방향

전체 레이아웃은 넓은 여백과 분명한 구역 구분을 중심으로 구성합니다. 작은 카드 여러 개를 과하게 쌓기보다, 큰 정보 블록을 사용해 메시지가 한눈에 들어오게 만듭니다.

### 기본 구성 원칙

- 중요한 메시지는 화면 위쪽에 크게 배치합니다.
- 한 구역에는 하나의 핵심 메시지만 담습니다.
- 배경색이 있는 컬러 블록은 넓게 사용합니다.
- 컬러 블록 사이에는 흰 배경 구간을 두어 숨 쉴 공간을 만듭니다.
- 제품 이미지와 문구는 서로 겹치지 않게 배치합니다.

## 6. 컬러 블록 사용법

이 스타일의 핵심은 큰 파스텔 컬러 블록입니다. 컬러 블록은 단순한 장식이 아니라, 내용의 흐름을 나누고 중요한 이야기를 강조하는 역할을 합니다.

### 사용 예시

- 상세페이지에서 제품 장점을 소개하는 구간
- 패키지 디자인에서 제품명과 핵심 문구가 들어가는 앞면
- SNS 이미지에서 이벤트 문구를 강조하는 배경
- 배너에서 할인, 신제품, 예약 안내를 보여주는 영역
- 라벨디자인에서 맛, 향, 용량, 제품군을 구분하는 색상 면

### 주의할 점

- 한 화면에 너무 많은 색상 블록을 동시에 넣지 않습니다.
- 컬러 블록 위의 글자는 반드시 잘 읽혀야 합니다.
- 색상은 제품 성격과 고객층에 맞게 선택합니다.
- 컬러 블록은 모서리를 부드럽게 처리하면 친근한 느낌이 강해집니다.

## 7. 버튼과 행동 유도 문구

버튼은 둥근 알약 형태를 기본으로 제안합니다. 버튼 모양을 통일하면 전체 디자인이 정돈되어 보이고, 사용자가 어떤 행동을 해야 하는지 쉽게 이해할 수 있습니다.

### 버튼 스타일

- **주요 버튼**: 검정 배경에 흰 글자
- **보조 버튼**: 흰 배경에 검정 글자
- **이벤트 버튼**: 강조 색상 배경에 흰 글자

### 문구 예시

- 구매하기
- 상담 신청
- 자세히 보기
- 혜택 확인
- 지금 확인하기
- 샘플 요청

## 8. 이미지 사용 방향

이미지는 실제 제품이나 서비스가 잘 보이는 것을 우선합니다. 분위기만 있는 이미지보다 고객이 무엇을 받게 되는지 바로 이해할 수 있는 이미지가 좋습니다.

### 작업별 이미지 방향

- **로고**: 작은 크기에서도 알아보기 쉬운 형태가 좋습니다.
- **배너**: 핵심 문구와 제품 이미지가 바로 보여야 합니다.
- **라벨디자인**: 제품명, 용량, 맛, 특징이 잘 구분되어야 합니다.
- **패키지 디자인**: 진열됐을 때 한눈에 브랜드와 제품군이 보여야 합니다.
- **상세페이지**: 제품 장점, 사용 방법, 후기, 구성품을 순서대로 보여줍니다.
- **SNS 이미지**: 첫 화면에서 멈춰 보게 만드는 짧은 문구와 선명한 이미지가 중요합니다.

## 9. 형태와 모서리

전체적으로 부드러운 모서리를 사용해 친근하고 현대적인 느낌을 만듭니다. 다만 모든 요소를 과하게 둥글게 만들면 가벼워 보일 수 있으므로, 제품 성격에 맞게 조절합니다.

### 권장 형태

- 작은 정보 박스: 살짝 둥근 모서리
- 큰 컬러 블록: 더 크게 둥근 모서리
- 버튼: 완전히 둥근 알약 형태
- 아이콘 버튼: 원형
- 제품 이미지 프레임: 적당히 둥근 사각형

## 10. 적용 예시

### 로고 디자인

단순하고 기억하기 쉬운 형태를 우선합니다. 복잡한 장식보다 브랜드 이름과 성격이 분명하게 보이는 방향이 좋습니다.

### 배너 디자인

큰 제목, 짧은 설명, 행동 유도 버튼을 중심으로 구성합니다. 배경은 흰색 또는 하나의 파스텔 컬러 블록을 사용합니다.

### 라벨디자인

제품명, 제품 종류, 용량, 특징이 빠르게 보이게 합니다. 맛이나 향이 여러 개인 제품은 컬러 블록으로 제품군을 구분할 수 있습니다.

### 패키지 디자인

매장 진열이나 택배 개봉 상황을 생각해 앞면의 정보 우선순위를 정합니다. 제품명과 핵심 장점은 멀리서도 읽히게 합니다.

### 상세페이지 디자인

상단에는 핵심 장점을 크게 보여주고, 아래로 내려가며 문제 제기, 제품 장점, 사용 방법, 후기, 구매 안내 순서로 정리합니다.

### SNS 이미지 디자인

짧은 문구와 강한 시각 요소를 사용합니다. 작은 화면에서도 글자가 잘 읽히도록 문장 수를 줄이고 여백을 충분히 둡니다.

## 11. 해야 할 것

- 흑백 중심의 기본 구조를 유지합니다.
- 강조 구간에는 하나의 큰 컬러 블록을 사용합니다.
- 제목은 짧고 크게, 본문은 쉽게 작성합니다.
- 버튼은 둥근 형태로 통일합니다.
- 제품 이미지와 핵심 문구가 먼저 보이게 합니다.
- 한 화면에 하나의 핵심 메시지만 담습니다.

## 12. 피해야 할 것

- 특정 서비스의 로고나 공식 사례를 그대로 사용하는 것
- 너무 많은 색상을 한 화면에 넣는 것
- 글자가 작거나 흐려서 읽기 어려운 것
- 컬러 블록 위에 대비가 약한 글자를 올리는 것
- 그림자와 장식을 과하게 넣는 것
- 버튼 모양을 섞어서 화면을 복잡하게 만드는 것
- 공식 디자인 시스템처럼 단정적으로 표현하는 것

## 13. 다음 확인 사항

실제 디자인 제작 전에는 아래 내용을 먼저 확인해야 합니다.

- 디자인할 제품이나 서비스 이름
- 주요 고객층
- 판매 또는 노출 채널
- 원하는 분위기
- 피하고 싶은 색상이나 스타일
- 참고 이미지 또는 경쟁 제품
- 필요한 결과물 종류와 크기
- 인쇄용인지 온라인용인지 여부
