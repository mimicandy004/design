# AI 디자인 제작 작업 안내

이 프로젝트는 미용 브랜드 **DADA**의 헤나 염색약 라벨 디자인을 준비하기 위한 작업 폴더입니다. 아직 실제 라벨 이미지를 만들기 전, 필요한 정보와 디자인 기준을 정리하는 단계입니다.

## 작업 원칙

- 이번 작업의 목적은 DADA 헤나 염색약 라벨 이미지를 만들기 위한 준비입니다.
- 브랜드명, 제품명, 컬러명, 용량, 주요 특징, 주의사항처럼 라벨에 꼭 필요한 정보를 먼저 확인합니다.
- 미용 제품답게 세련되고 신뢰감 있는 방향을 유지합니다.
- 헤나 염색약의 자연스러운 느낌이 잘 드러나도록 색상, 문구, 이미지 방향을 정리합니다.
- 초보자도 이해할 수 있는 쉬운 말로 설명합니다.
- 특정 브랜드의 공식 디자인 시스템이라고 단정하지 않습니다.
- 참고 브랜드나 참고 이미지는 방향을 잡기 위한 자료로만 사용하고, 그대로 따라 하지 않습니다.
- 실제 디자인 파일 제작은 사용자의 확인을 받은 뒤 진행합니다.

## 기본 작업 흐름

1. `context/business-context.md`에서 DADA 브랜드와 헤나 염색약 제품 정보를 확인합니다.
2. `context/design-brief.md`에서 이번 라벨 디자인의 목적, 필요한 정보, 크기, 형식을 확인합니다.
3. `brand/DESIGN.md`에서 라벨에 적용할 임시 디자인 방향을 참고합니다.
4. 참고할 제품 사진, 로고, 기존 라벨, 경쟁 제품 이미지는 `assets/` 폴더에 모읍니다.
5. 완성된 라벨 이미지와 최종 결과물은 `output/` 폴더에 정리합니다.

## 라벨 디자인에서 우선 확인할 것

- DADA 로고 또는 브랜드 표기 방식
- 헤나 염색약의 정확한 제품명
- 컬러명 또는 컬러 번호
- 제품 용량
- 라벨 실제 크기
- 온라인용인지 인쇄용인지 여부
- 필수 표기 문구와 주의사항
- 참고하고 싶은 라벨 이미지

## 디자인 방향

- 자연스럽고 깨끗한 미용 제품 이미지를 우선합니다.
- 온라인 썸네일에서도 브랜드명과 제품명이 잘 보이게 합니다.
- 매장 진열 시 컬러 구분이 쉬운 라벨 구조를 고려합니다.
- 글자가 너무 많아 보이지 않도록 정보 우선순위를 정합니다.
- 파스텔 컬러 블록과 흑백 중심의 깔끔한 구성을 참고하되, DADA 제품에 맞게 조정합니다.

## 멈춤 기준

문서와 폴더를 정리한 뒤에는 실제 라벨 이미지 제작으로 바로 넘어가지 않습니다. 사용자의 확인을 받은 뒤 다음 작업을 진행합니다.
