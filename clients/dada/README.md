# DADA 작업 폴더

이 폴더는 DADA 미용 브랜드의 헤나 염색약 라벨 작업 자료를 보관합니다.

## 파일 역할

- `business-context.md`: DADA 브랜드와 제품 정보를 정리한 문서
- `design-brief.md`: DADA 헤나 염색약 라벨 제작 요청서
- `data.json`: 템플릿에 넣을 DADA 전용 제품 데이터
- `assets/`: DADA 참고 이미지와 원본 목업 자료를 로컬에 보관하는 폴더
- `output/`: DADA 라벨 시안과 목업 결과물을 로컬에 보관하는 폴더

## 사용 템플릿

- `templates/henna-label/`

## 다음 업체 작업 방법

다음 업체가 생기면 이 폴더를 그대로 복사하지 말고, `templates/henna-label/data-example.json`을 참고해서 새 `clients/<업체명>/` 폴더를 만듭니다.

## GitHub 보관 기준

이미지 시안과 참고 이미지는 기본적으로 GitHub에 올리지 않습니다. 최종 공유가 필요한 작은 미리보기만 `preview/` 폴더를 만들어 넣습니다.
